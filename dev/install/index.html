Installation

Install Julia

Download the Current stable release of julia.

On windows, install and click on julia.exe (likely under <USERNAME>\AppData\Local\Programs\Julia-1.8.5\bin\) to start The Julia REPL.

Add additional registry

TWAISWCF and some of its dependencies are registered on OkRegistry; thus, you have to add OkRegistry to your environment firstly as follows.

Open julia REPL, switch to Pkg REPL and:

(@v1.8) pkg> registry add https://github.com/okatsn/OkRegistry.git
Hint
  • Pkg comes with a REPL. Enter the Pkg REPL by pressing ] from the Julia REPL. See Pkg for more information.
  • Enter ←Backspace button to switch back to julia REPL (julia>).

Add TWAISWCF

In your project environment, add TWAISWCF:

(@v1.8) pkg> add TWAISWCF
Hint

It can but it is in general not recommended to work on the (@1.X) julia> environment. See the followings to learn how to generate a project and activate the project environment.

  • Working with Environments
  • How to setup Project Environments in Julia
  • Create a temporary/throwaway environement

Keep updated to the latest version

(@v1.8) pkg> up
